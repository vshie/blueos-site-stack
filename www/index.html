ESPHome snippet

mqtt:
  broker: <blueos-ip>
  # discovery: false
  topic_prefix: blueos/relay

Replace <blueos-ip> with the Pi address (or hostname) on your LAN. Sensors/switches auto-ingest into Influx.

Other BlueOS extensions (telemetry / commands)

Publish under blueos/ext/<your-extension-slug>/…:

blueos/ext/<slug>/<metric>/state   numeric telemetry (auto-ingested)
blueos/ext/<slug>/json             JSON telemetry, multiple fields (auto-ingested)
blueos/ext/<slug>/cmd/<action>/set  commands (your extension subscribes)

See the README "Topic conventions" section for the full spec.
