Clock without internet

When the Pi can’t reach internet time servers, Site Stack can read the ESP board’s battery-backed clock so logs stay in the right order. Site UI shows this as the “Time” pill (NTP vs ESP clock).

Advanced — message names (for developers)

Boards publish under blueos/<device>/…. Other BlueOS extensions can use blueos/ext/<name>/…. Full details are in the README. 

ESPHome sensors:  blueos/relay/sensor/…/state
ESPHome relays:   blueos/relay/switch/relay_1/state  (+ /command)
Extension metric: blueos/ext/mysensor/temp/state
Extension JSON:   blueos/ext/mysensor/json
